### 购物车空组件

> 购物车空状态组件

#### 基本调用
``` 
 <shop-empty title="暂无商品" btn-text="按钮文字" bind:btn-click="gotoshop"/>
```

#### 属性 Props

|   参数   |   说明   |  类型  | 可选值 | 默认值 |  可选性  |
| :------: | :------: | :----: | :----: | :----: | :------: |
|  title   |   标题   | String |   -    |   -    |    -     |
| btn-text| 按钮文字 | String |   -    |   -    | Required |

#### 事件

| 参数  |     说明     |   类型   | 可选值 | 默认值 | 可选性 |
| :---: | :----------: | :------: | :----: | :----: | :----: |
| ontap | 按钮点击事件 | Function |   -    |   -    |   -    |
