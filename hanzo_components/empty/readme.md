###  无内容提示组件

#### 适用场景
> 列表无商品、无订单提示语

#### 基本调用
``` 
 <empty></empty>
```
#### 参数 props
| 参数 |   说明    |  类型  | 可选值 | 默认值 | 可选性 |
| :--: | :-------: | :----: | :----: | :----: | :----: |
| type | Icon 名称 | String |   -    |   -    |   -    |
| img  | 图片地址  | String |   -    |   -    |   -    |

####  插槽 slot
| 名称          |  说明               | 备注            |
| ------------- |:-------------:| :--------------------| 
| desc          | 自定义描述(文本、按钮、事件等)  |                 |
