### 纯文本空组件

> 搜索页、订单页、评价页空状态

#### 基本调用
``` 
 <brand-empty title="抱歉,未搜索到“肉包子”相关的商品" desc="请换一个关键词试试"/>
```

#### 属性 Props

| 参数  |   说明    |  类型  |    可选值     | 默认值 | 可选性 |
| :---: | :-------: | :----: | :-----------: | :----: | :----: |
| type  | Icon 名称 | String | list / search |  list  |   -    |
| title |   标题    | String |       -       |   -    |   -    |
| desc  |   描述    | String |       -       |   -    |   -    |
