### 菜单组件

#### API

| 参数        | 类型   | 描述           | 默认值   |
| ----------- | ------ | -------------- | -------- |
| menuList    | Array  | 渲染菜单的数组 | -        |
| align       | String | 布局方式       | vertical |
| height      | String | 高度           | -        |
| selectIndex | Number | 选中项         | 0        |

#### 事件

| 事件名     | 描述       | 参数 |
| ---------- | ---------- | ---- |
| changeMenu | 切换时出发 | data |
